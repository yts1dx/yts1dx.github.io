Use of Flex

Aug 15th, 2016 8:19 pm

Format of the Input File

Input File由三部分组成，通过%%进行隔离

definitions %% rules %% user code

definitions

该部分用来定义Name，来指定一种匹配的形式;或者指定start condition，通过start condition, flex可以使得一下规则在某些场景下才能生效。

rules

定义pattern和action

Flex global Macro

yytext & yyleng

yytext:指向匹配内容的指针；可以通过声明来指定yytext 是一个pointer还是一个array。 yyleng:匹配内容的长度

yyextra

通过yyextra可以存放用户指定的数据。在scanner之外，可以通过yyset_extra和yyget_extra来设置和货到yyextra的内容。 默认情况下yyextra为void * 类型，但是用户可以重新定义其类型。

Flex global Functions

yylex()

当yylex()被调用时，其读取输入文件，直到某个action执行return或到达文件尾部为止。
